Основы ООП

Похожие работы

  • Social Library (Symfony)

Проект "Социальная библиотека"


Основные задачи:

  • хранение информации о прочитанных книгах;
  • обмен отзывами с другими пользователями;
  • отслеживание новых поступлений.

Необходимо:

  • проанализировать требования и выделить основные сущности (UML -> Class Diagram);
  • отобразить данные взаимоотношения с помощью Entity-Relationship Diagram (ERD);
  • описать сущности на JS (потренироваться с реализацией ООП в JS).
