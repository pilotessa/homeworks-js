ECMAScript 6

Деструктурирование


Существует некий объект вида: 

var obj = {
    name: "John",
    middleName: undefined,// you may delete this field. Just to have a point where it's placed.
    surname: "Smith",
    identity: "123456790",
    additionalData: {
        hobbies: ["Coffee Roasting", "Acting", "Coloring"],
        address: {
            country: "Australia",
            region: {
                name: "2000 New South Wales"
            },
            street: {
                name: "Farrer Place",
                buildNumber: "1"
            },
            city: undefined, // you may delete this field. Just to have a point where it's placed.

        },
        expirience: undefined, // you may delete this field. Just to have a point where it's placed.
    },
    rating: {
        average: "4.8",
        votes: [
            {
                mark: "5"
            },
            {
                mark: "5"
            },
            {
                mark: "5"
            },
            {
                mark: "5"
            },
            {
                mark: "4"
            }
        ]
    }
}

Необходимо извлечь данные:

  • fistname -- obj’s name
  • middleName -- obj’s optional middleName. Default value: ‘N/A’
  • lastname -- obj’s surname
  • identity -- obj’s identity
  • votes -- obj’s rating votes array... Default value: empty array
  • mainHobby -- obj’s first element in hobbies array. Default value: ‘N/A’
  • otherHobbies -- obj’s rest hobbies except the first one.
  • country -- obj’s country from address
  • city -- obj’s city from address. Default value: ‘Sydnay’
  • region -- obj’s region name.
  • streetName -- obj’s street name
  • streetNumber -- obj’s street number
